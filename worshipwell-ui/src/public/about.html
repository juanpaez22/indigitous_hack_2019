I came up with the name of the site. Along the way, I learned basic frontend concepts in html and JavaScript as well as basic backend in Python. I created the Flask-based web API that generated the list of songs based on the requested theme that would then be presented to the user. I created the database of songs that we use to obtain data from. I learned how to interface with Flask and became exposed to backend development with the Flask framework. I learned about the division of labor between front-end and back-end, and also the ways in which these areas connect. I also got to work more with python and work with the html for the about page.
